# Documentation

This folder contains user guides, technical notes, and a living specification for Music Downloader.

Structure
- specs.md — product and technical specifications (living document)
- technical/
  - architecture.md — system architecture and components
  - api.md — API overview and conventions
  - security.md — security practices and secrets handling
- user/
  - getting-started.md — installation and usage
  - faq.md — common issues and solutions

Quick links
- API Swagger UI (runtime): http://localhost:8000/api/docs
- Redoc (runtime): http://localhost:8000/api/redoc

Contributing to docs
- Keep documents in clear English.
- Update docs alongside code changes.
- Reference endpoints with their full /api/... path.
